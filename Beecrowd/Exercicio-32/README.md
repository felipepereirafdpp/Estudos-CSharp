# 📚 Desafio Beecrowd - 32

**📅 Data:** 14/07/2026  
**⏱️ Tempo do Desafio:** 8 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê duas matrizes de números inteiros com a mesma quantidade de linhas e colunas, calcula a soma dos elementos correspondentes e gera uma terceira matriz com os resultados, exibindo-a ao final.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Matrizes (`int[,]`).
* Estrutura de repetição `for`.
* Percorrer linhas e colunas de uma matriz.
* Operações entre matrizes.
* Soma de elementos correspondentes.
* Armazenamento de resultados em uma terceira matriz.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização do método `Split()` para separar os dados informados pelo usuário.
