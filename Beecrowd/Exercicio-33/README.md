# 📚 Desafio Beecrowd - 33

**📅 Data:** 14/07/2026  
**⏱️ Tempo do Desafio:** 15 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê uma matriz quadrada de números reais e realiza diversas operações: calcula a soma dos elementos positivos, exibe os elementos de uma linha e de uma coluna informadas pelo usuário, imprime a diagonal principal e altera a matriz elevando ao quadrado todos os valores negativos.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Matrizes (`double[,]`).
* Estrutura de repetição `for`.
* Percorrer linhas e colunas de uma matriz.
* Soma de elementos positivos.
* Acesso a uma linha e a uma coluna específicas.
* Identificação e impressão da diagonal principal (`i == j`).
* Alteração de elementos da matriz com estrutura condicional `if`.
* Operações matemáticas (elevação ao quadrado).
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização de `CultureInfo.InvariantCulture` para leitura e exibição de números reais.
* Utilização do método `Split()` para separar os dados informados pelo usuário.
