# 📚 Desafio Beecrowd - 26

**📅 Data:** 12/07/2026
**⏱️ Tempo do Desafio:** 12 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê a altura e o sexo de um grupo de pessoas, identifica a menor e a maior altura, calcula a média das alturas das mulheres e informa a quantidade de homens presentes no grupo.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Vetores (`double[]` e `string[]`).
* Estrutura de repetição `for`.
* Estruturas condicionais (`if`).
* Comparação de valores para encontrar o maior e o menor elemento.
* Cálculo de média aritmética.
* Acumulação de valores e contagem de elementos.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização de `CultureInfo.InvariantCulture` para leitura e exibição de números decimais.
* Utilização do método `Split()` para separar os dados informados pelo usuário.
