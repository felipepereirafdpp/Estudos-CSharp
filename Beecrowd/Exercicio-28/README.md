# 📚 Desafio Beecrowd - 28

**📅 Data:** 14/07/2026
**⏱️ Tempo do Desafio:** 5 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê as dimensões de uma matriz de números inteiros, armazena seus valores e exibe somente os números negativos encontrados na matriz.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Matrizes (`int[,]`).
* Estrutura de repetição `for`.
* Percorrer linhas e colunas de uma matriz.
* Estrutura condicional `if`.
* Identificação de valores negativos.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização do método `Split()` para separar os dados informados pelo usuário.
