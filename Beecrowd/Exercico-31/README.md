# 📚 Desafio Beecrowd - 31

**📅 Data:** 14/07/2026  
**⏱️ Tempo do Desafio:** 12 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê uma matriz quadrada de números inteiros e calcula a soma dos elementos localizados acima da diagonal principal, exibindo o resultado ao final.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Matrizes (`int[,]`).
* Estrutura de repetição `for`.
* Percorrer linhas e colunas de uma matriz.
* Identificação de elementos acima da diagonal principal (`j > i`).
* Acumulação de valores.
* Estrutura condicional `if`.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização do método `Split()` para separar os dados informados pelo usuário.
