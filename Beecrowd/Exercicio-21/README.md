# 📚 Desafio Beecrowd - 21

**📅 Data:** 10/07/2026
**⏱️ Tempo do Desafio:** 10 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê a quantidade de elementos dos vetores, recebe os valores de dois vetores inteiros e gera um terceiro vetor contendo a soma dos elementos correspondentes de cada posição. Ao final, o programa exibe o vetor resultante.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Vetores (`int[]`).
* Estrutura de repetição `for`.
* Percorrer vetores utilizando índices.
* Operações entre elementos de vetores.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização do método `Split()` para separar os valores informados pelo usuário.
