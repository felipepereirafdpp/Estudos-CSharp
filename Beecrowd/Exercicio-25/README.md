# 📚 Desafio Beecrowd - 25

**📅 Data:** 12/07/2026
**⏱️ Tempo do Desafio:** 5 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê o nome e as notas de vários alunos, calcula a média de cada um e exibe apenas os nomes daqueles que foram aprovados, considerando média maior ou igual a 6,0.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Vetores (`string[]` e `double[]`).
* Estrutura de repetição `for`.
* Estruturas condicionais (`if`).
* Cálculo de média aritmética.
* Manipulação de números decimais.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização de `CultureInfo.InvariantCulture` para leitura de valores decimais.
* Utilização do método `Split()` para separar os dados informados pelo usuário.
