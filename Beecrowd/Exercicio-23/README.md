# 📚 Desafio Beecrowd - 23

**📅 Data:** 10/07/2026
**⏱️ Tempo do Desafio:** 10 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê a quantidade de elementos de um vetor de números inteiros, identifica quais valores são pares, calcula a média aritmética apenas desses números e exibe o resultado formatado com duas casas decimais.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Vetores (`int[]`).
* Estrutura de repetição `for`.
* Estruturas condicionais (`if`).
* Operador de módulo (`%`) para verificar números pares.
* Cálculo de média aritmética.
* Acumulação de valores e contagem de elementos.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização de `CultureInfo.InvariantCulture` para exibição de números decimais.
* Utilização do método `Split()` para separar os valores informados pelo usuário.
