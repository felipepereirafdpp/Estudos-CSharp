# 📚 Desafio Beecrowd - 29

**📅 Data:** 14/07/2026
**⏱️ Tempo do Desafio:** 8 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê uma matriz quadrada de números inteiros e calcula a soma dos elementos de cada linha, exibindo o resultado correspondente para cada uma delas.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Matrizes (`int[,]`).
* Estrutura de repetição `for`.
* Percorrer linhas e colunas de uma matriz.
* Acumulação de valores.
* Reinicialização de variável acumuladora.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização do método `Split()` para separar os dados informados pelo usuário.
