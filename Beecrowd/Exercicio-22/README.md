# 📚 Desafio Beecrowd - 22

**📅 Data:** 10/07/2026
**⏱️ Tempo do Desafio:** 10 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê a quantidade de elementos de um vetor de números reais, calcula a média aritmética de todos os valores informados e, em seguida, exibe apenas os elementos que estão abaixo da média calculada.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Vetores (`double[]`).
* Estrutura de repetição `for`.
* Cálculo de média aritmética.
* Operações com números decimais.
* Estruturas condicionais (`if`).
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização de `CultureInfo.InvariantCulture` para leitura e exibição de números decimais.
* Utilização do método `Split()` para separar os valores informados pelo usuário.
