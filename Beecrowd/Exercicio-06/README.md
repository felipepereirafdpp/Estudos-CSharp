# 📚 Desafio Beecrowd - 06

**📅 Data:** 03/07/2026
**⏱️ Tempo do Desafio:** 15 Minutos

## 📝 Resumo do Exercicio

>  Desenvolvi um programa que calcula a quantidade mínima de caixas necessárias para embalar um determinado número de produtos, utilizando caixas de diferentes capacidades (100, 50, 20, 10, 5 e 1).

---

## 💡 Conceitos aprendidos

- Leitura de dados.
- Operações aritméticas.
- Divisão inteira (/).
- Operador de resto (%).
- Lógica para decomposição de valores.
- Resolução de problemas utilizando cálculos sequenciais.
