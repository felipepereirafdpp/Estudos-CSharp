# 📚 Desafio Beecrowd - 30

**📅 Data:** 14/07/2026  
**⏱️ Tempo do Desafio:** 6 Minutos

## 📝 Resumo do Exercício

> Desenvolvi um programa que lê uma matriz quadrada de números inteiros e identifica o maior elemento de cada linha, exibindo o maior valor encontrado em cada uma delas.

## 💡 Conceitos aprendidos

* Leitura de dados.
* Matrizes (`int[,]`).
* Estrutura de repetição `for`.
* Percorrer linhas e colunas de uma matriz.
* Comparação de valores.
* Estrutura condicional `if`.
* Utilização de variável auxiliar para armazenar o maior elemento.
* Entrada e saída de dados (`Console.ReadLine` e `Console.WriteLine`).
* Utilização do método `Split()` para separar os dados informados pelo usuário.
